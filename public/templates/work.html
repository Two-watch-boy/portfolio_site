@SomePoint

@SomePoint is a day optimization tool developed to help people find the fastest route to compleet a set of tasks, not nessasarly time sensitive, but that should all be done by the day's end.

Built with Ruby on Rails, PostgreSQL, HTML, CSS, and javascript.

A project with John Ruhsam, Jonathan Ortiz, and Justin Pettit.

HandPrint

Handprint is a travel blog for leaving reviews on cities, posting comments, and showing pictures.

Built with Ruby on Rails, PostgreSQL, HTML, CSS, and javascript.

A project with Arthur Tonelli, Sarah Bannister, and Joanne Abad.

Duck Derby

Ducky Derby is a simple racing game where two people can race their ducks across the screen using the "F" and "J" keys.

Built with HTML, CSS, and javascript, and Jquery.
